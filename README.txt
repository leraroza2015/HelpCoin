# HelpCoin
create the folder HelpCoin in the folder "%APPDATA%\
to do this, use the tap win+r
use the command       %APPDATA%

after creating the HelpCoin folder, upload the downloaded archive 
in files helpcoin.conf and mine.bat it is necessary to replace the label /you-password/ past your password
and login user
to open files for this type, use a special program or hosting

Congratulations, you are ready to go!
To open the wallet, use helpcoin-qt
Start mining, use the mine file.bat
